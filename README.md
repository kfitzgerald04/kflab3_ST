# CSCI 4325 Software Testing Lab 3 

Install/Run Instructions

1. Clone the repository (copy/paste in IntelliJ)
2. 'cd' in with repo name (optional, but make sure you're in the directory with the pom.xml file)
3. Build the project (mvn clean install) --optional
4. Run the project (command line | mvn test OR select 'Run" for the StayPriceCalculatorTest.java class)
